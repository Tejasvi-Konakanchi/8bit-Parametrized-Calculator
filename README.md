# SoC Knowledge Base

A personal collection of notes documenting my journey towards becoming a System-on-Chip (SoC) Engineer.

The goal of this repository is to:
- Learn the digiral design fundamentals
- Document important concepts in my own words
- Collect useful diagrams, guides, and code snippets
- Build a long-term reference for interviews, projects and future work
